E-Commerce Storefront · GetPortalConfiguration API
للمصمم (CSS/SCSS) — بدون الحاجة لمعرفة TypeScript

القيم تأتي من الباك إند تلقائياً وتُحقَن على :root. في ملفات SCSS استخدم var(--portal-*) بدلاً من الألوان الثابتة. إذا فشل تحميل الـ API تُستخدم قيم Default أدناه.

1. الألوان والفونتات — من الـ API

حقل الـ API متغير CSS Default مثال من الباك الاستخدام
primaryColor --portal-primary #775a19 #b1d822 لون رئيسي، hover، links، accent
secondaryColor --portal-secondary #4e5f7c #ffffff عناصر ثانوية
accentColor --portal-accent #d24858 #ffc107 تمييز، badges، تنبيهات
backgroundColor --portal-background #f9f9f7 #ffffff خلفية الصفحة
textColor --portal-text #1a1a1a #000000 لون النص الأساسي
headerColor --portal-header-bg #ffffff #ffffff خلفية الهيدر / الـ nav
footerColor --portal-footer-bg #775a19 #000000 خلفية الفوتر
fontFamilyAr --portal-font-family-ar Manrope, system-ui, … Cairo فونت الواجهة العربية
fontFamilyEn --portal-font-family-en Manrope, system-ui, … Roboto فونت الواجهة الإنجليزية
fontSizeBase --portal-font-size-base 16px 14px حجم النص العادي
fontSizeHeading --portal-font-size-heading 30px 24px حجم العناوين
fontSizeSmall --portal-font-size-small 14px 12px نص صغير / captions
