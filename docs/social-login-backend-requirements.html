1) Google Cloud Console

  • فتح OAuth Client من نوع Web application (نفس ClientId أعلاه)
  • المسار: APIs & Services → Credentials
  • إضافة في Authorized JavaScript origins (بدون / في الآخر):
http://localhost:4200
http://compassint.ddns.net:PORT   ← حسب بورت الفرونت الفعلي
https://DOMAIN   ← دومين الإنتاج

Google Sign-In يحتاج JavaScript origins — ليس Redirect URIs فقط.

2) Facebook Developers

  • إضافة المختبرين كـ Testers إذا التطبيق في Development mode
  • ضبط App Domains و Facebook Login لـ localhost ودومين السيرفر/الإنتاج

3) تأكيد على السيرفر

  • Google.ClientId = القيمة أعلاه
  • Facebook.AppId = القيمة أعلاه
  • POST /api/TokenAuth/ExternalAuthenticateECommerce يعمل مع header Abp.TenantId
